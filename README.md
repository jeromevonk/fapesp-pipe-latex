# fapesp-pipe-latex

Este é um possível modelo **(não oficial**) para você utilizar na elaboração do Projeto de Pesquisa para o PIPE da FAPESP.

Você pode se inspirar nele se for escrever o seu projeto de pesquisa em LATEX.

Disclaimer: na página [das normas do PIPE](https://fapesp.br/pipe/normas/), a FAPESP fornece um template em .docx. Eu apenas fiz um modelo parecido em LATEX porque me facilita a digramação.

Modelo feito em Novembro de 2024.

Se quiser usar e não sabe por onde começar:

* Entre no [Overleaf](https://www.overleaf.com/)
* Copie e cole o código do arquivo *main.tex*
* Crie o arquivo *refs.bib* e copie o conteúdo
* Compile
